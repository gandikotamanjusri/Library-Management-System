# Library-Management-System
# Project Overview

This is a console-based Library Management System developed using Java and Object-Oriented Programming (OOP) principles.
The system allows users to manage books and members, perform search operations, and handle borrowing and returning of books through a menu-driven interface.

The project demonstrates:

Encapsulation
Class relationships
Real-world object modeling
Input validation
Menu-based interaction

# Technologies Used

Java (JDK 8 or higher)
Java Collections (ArrayList)
Scanner (for user input)

# Features

Add new books
Register library members
Display all books
Display available books
Search books by title/author/genre
Borrow books with availability validation
Return borrowed books
Menu-driven user interface
